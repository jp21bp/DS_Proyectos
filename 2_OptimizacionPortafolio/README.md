# Optimizando Riesgos de Portafolios Financieros

Este proyecto explorara diferentes acciones para crear y optimizar un portafolio financiero. 

## Convirtiendo problema empresarial en problema de Ciencias de Datos (CD)
Existen 5 componentes para transformar un problema empresarial en problema CD:

* Contexto del Problema:
    - Los portafolios financiero conllevan alto riesgo impredecible.

* Objetivo:
    - Crear una herramienta de que pueda reducir el riesgo de un portafolio.

* Datos:
    - Utilizar acciones selecionadas del S&P 500

* Restricciones:
    - No existen limitaciones concretas

* Metricas:
    - Reducir la volatilidad de rentabilidad de un portafolio


## Pasos lanzados
Existen 8 pasos dentro un proyecto de CD:

1. Definir el Problema
2. Coleccionar Datos
3. Preparar y Limpiar Datos
4. EDA (Exploratory Data Analysis)
5. Modelar Datos
6. Evaluar Modelo
7. Comunicar Resultados
8. Desplegar Modelo
