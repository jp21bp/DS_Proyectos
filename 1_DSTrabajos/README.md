Componentes adicionales de toque personal:

* XAI
    - Despues de entrenar todos los modelos, voy hacer XAI para profundizar la evaluacion de todos los modelos

## Pasos:

Este proyecto seguira los siguente pasos en su desarrollo:
1. Definir Problema
2. Coleccion de Datos
3. Limpiar datos
4. EDA
5. Crear modelos
6. Desplegar Modelos

## Formulacion del Problema (Paso 1)

Este paso va a transformar un problema de negocio en un problema de DS:
* Context del Negocio (Imaginado):
    - Hay un negocio que quiere hacer postulaciones de trabajo para un DS
    - Quieren postular un sueldo que se refleja en el mercado a traves otras postulaciones similare
    - Al obtener un sueldo del mercado se van a mantener competitivo sin hacer gaastos inecesarios
* Objetivo:
    - Investigar el mercado laboral para establecer un sueldo adecuado para un DS dentro del negocio
* Disponibilidad de datos:
    - Se va a necesitar hacer un extraccion de otros trabajos en linea
* Restricciones:
    - Ninguna
* Metricas:
    - La creacion de un modelo que disponga un sueldo considerando diferente caracteristicas del trabajo

## Coleccion de datos (Paso 2)
